Introdução

Bem-vindo à documentação do projeto! Este documento foi criado para ajudá-lo a entender e usar o sistema.

Instalação

Siga os passos abaixo para instalar o projeto:

  1. Clone o repositório.
  2. Instale as dependências.
  3. Configure as variáveis de ambiente.
  4. Execute o projeto.

Referência da API

POST /v1/auth/signUp

Cadastra o usuário

Exemplo do da requisição

{
  "name": "thalles",
  "email": "thalles@gmail.com",
  "password": "#PasswordExample123"
}

        
Exemplo de resposta:
{
  "data": {
    "accessToken": {
      "token": "eyJhbGciOiJIU...",
      "duration": "24h",
      "expiresInMilliseconds": 86400000,
      "expiresInUnixEpoch": 1736184217364
    },
    "user": {
      "id": "fa1e76d5-39ca-4786-a0ba-dc5f25e33c97",
      "name": "thalles",
      "email": "thalles@gmail.com",
      "role": "USER",
      "deletedAt": null,
      "createdAt": "2025-01-05T17:23:37.356Z",
      "updatedAt": "2025-01-05T17:23:37.356Z"
    }
  },
  "href": {
    "url": "http://localhost:3000/v1/user",
    "method": "GET"
  }
}
